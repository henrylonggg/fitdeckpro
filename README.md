# FitDeck Pro

## Run locally
```bash
npm install
npm start
```

Open http://localhost:3000

## Render
- Service: Web Service
- Build command: `npm install`
- Start command: `npm start`
- Health path: `/health`

Features include 52-card workouts, coin-selected exercises, swipe completion, hydration/activity scoring, per-exercise rep analytics, and a daily muscle heatmap.
